ProTypist is a fully featured typing game. With a multiplayer mode,
custom games, quotes, top most common words in 7 languages and
multiple themes.

## Tech Stack

### Frontend

- React
- Redux
- SocketIO Client

### Backend

- NodeJS
- Express
- SocketIO
- MongoDB / Mongoose
